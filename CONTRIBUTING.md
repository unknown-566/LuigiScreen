# Contributing

Bug reports, reproducible test cases and documentation corrections are
welcome through GitHub Issues.

LuigiScreen is not currently accepting external code contributions. Please do
not submit pull requests containing source-code changes unless the project
owner has explicitly requested them.

This keeps copyright ownership clear while the Free and future Plus editions
are being designed. This policy may be replaced by a contributor agreement in
the future.

Before sharing diagnostics, follow the security guidance in `SECURITY.md` and
remove credentials, stream keys and private network information.
