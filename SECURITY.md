# Security Policy

## Supported versions

LuigiScreen is currently in alpha. Only the latest published alpha receives
security fixes.

## Reporting a vulnerability

Do not open a public issue containing:

- RTMP passwords or stream keys
- Public IP addresses that should remain private
- Server credentials
- Private logs or configuration files

Open a minimal GitHub issue asking for a private contact channel, without
including the sensitive details. The project owner will provide the next
reporting step.

## Logs

LuigiScreen masks common credentials in its own status output and logs, but
administrators must still review all files before sharing them publicly.
